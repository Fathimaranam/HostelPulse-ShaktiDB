# HostelPulse

## Title
HostelPulse – AI-Powered Occupancy and Meal Demand Forecasting Platform using ShaktiDB

## Problem Statement

In institutions where a central kitchen serves multiple hostels, food preparation is often based on rough estimates. During weekends, holidays, and exam periods, student occupancy changes significantly, resulting in food wastage or food shortages.

## Proposed Solution

HostelPulse allows students to:

- Apply leave
- Update return dates
- Set meal preferences

The system stores data in ShaktiDB and predicts meal demand based on hostel occupancy.

## Features

- Student Management
- Leave Management
- Meal Preferences
- Occupancy Tracking
- Food Demand Forecasting
- Waste Reduction Analytics

## Technology Stack

- ShaktiDB
- PostgreSQL
- Ubuntu 24.04 (WSL)
- GitHub
- Python

## Expected Outcome

Reduce food wastage and improve kitchen planning using real-time occupancy data.
